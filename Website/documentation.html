Unsupported Chromium Browser

If you are using Edge, Chrome or other Chromium based Browsers

Download

Download the Extension here

Installation

Unzip the downloaded file

In your Browser open: chrome://extensions/

Locate the Developer mode switch and activate it

Click Load unpacked extension and selected the unzipped folder

Now the extension is installed. Just visit Omegle and start using it
